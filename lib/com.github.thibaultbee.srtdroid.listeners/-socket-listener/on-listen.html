onListen

abstract fun onListen(ns: Socket, hsVersion: Int, peerAddress: InetSocketAddress, streamId: String): Int
Content copied to clipboard

Called to handle the incoming connection on the listening socket (and is about to be returned by srt_accept), but before the connection has been accepted.

See Also:srt_listen_callback

Return

return 0, if the connection is to be accepted. If you return -1, this will be understood as a request to reject the incoming connection.

Parameters

ns

the new SRT socket of the new connection

hsVersion

the handshake version

peerAddress

the address of the incoming connection

streamId

the value set to SockOpt.STREAMID option set on the peer side
